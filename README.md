## Drawing Components
### (a) Floor Plan 
The floor plan shows the typical layout of one floor of CT Tower.

**Features included:**
- Structural walls with correct line weights
- Doors with swing directions shown
- Windows along exterior walls
- Staircase on the left end of the building
- Elevator/shaft on the right end of the building
- Corridor running centrally along the full length of the floor
- Office rooms on both sides of the corridor
- Washroom/toilet blocks with fixtures (WCs and sinks)

**Layers Used:**

| Layer | Description |
|-------|-------------|
| WALLS | Structural and partition walls |
| DOORS | Door blocks with swing arcs |
| WINDOWS | Window openings along walls |
| STAIRS | Staircase layout |
| DIMENSIONS | All dimension lines and text |
| TEXT | Room labels and annotations |

### (b) Electrical Wiring Layout
The electrical plan shows the full wiring distribution across the floor.

**Features included:**
- Main electrical panels: LP-A-1, LP-A-2, LP-A-3, LP-B-1, LP-B-2, LP-B-4
- Ceiling light fixtures in every room and along the corridor
- Wall outlets positioned along room walls
- Single pole switches and 2-way switches at room entry points
- Wiring routes connecting panels to all lighting and socket points

**Symbols Legend:**

| Symbol | Meaning |
|--------|---------|
| Cross with square | Wiring point |
| Hourglass (yellow) | Electrical Panel |
| Rectangle (yellow) | Ceiling Light |
| Circle with line | Wall Outlet |
| $ symbol | Single Pole Switch |
| $₃ symbol | 2-Way Switch |

### (c) Water Supply & Plumbing Plan
The plumbing plan shows the complete water supply and drainage system.

**Features included:**
- Cold Water Riser (CW RISER) from rooftop tank
- Horizontal soil pipes (green, 0.50mm) serving all WC fixtures
- Horizontal waste pipes (green, 0.35mm) serving washbasins and sinks
- Soil Stack/Vent connections to sewer and vent
- Clean-out (CO) access points at key locations

**Symbols Legend:**

| Symbol | Meaning |
|--------|---------|
| Blue circle with X | Cold Water Riser |
| Green rough circle (large) | Soil Stack/Vent — 0.50mm |
| Green rough circle (small) | Waste Stack/Vent — 0.35mm |
| Blue line | Water Pipe |
| Green line (thick) | Soil Pipe — 0.50mm |
| Green line (thin) | Waste Pipe — 0.35mm |
| CO | Clean Out |

### (d) Wi-Fi & Network Cabling Plan
The network plan shows complete Wi-Fi coverage and data cabling.

**Features included:**
- Network Tray (cyan) as the backbone along the full corridor
- Network Racks at two distribution points
- Wi-Fi Access Points at regular intervals
- AP Coverage Range circles — full floor coverage
- RJ45 Data Outlets in each room

**Symbols Legend:**

| Symbol | Meaning |
|--------|---------|
| Orange/red rectangle | Network Rack |
| Cyan thick line | Network Tray |
| Blue line | Data Cable |
| Solid green circle | Access Point |
| Small square (magenta) | RJ45 Outlet |
| Dashed green circle | AP Coverage Range |
